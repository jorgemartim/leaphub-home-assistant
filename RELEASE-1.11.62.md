# Leap Hub Gateway 1.11.62

## Contrato visual versão 6

O Gateway passa a enviar um mapa explícito dos componentes visuais. Cada sinal conhecido é enviado como verdadeiro ou falso; sinais que a API não disponibilizou ficam listados como desconhecidos e nunca são tratados como fechados.

O contrato inclui portas, porta-malas, vidros, teto, cortina, luzes, pisca-alerta, sentinela, climatização, pré-aquecimento, carga concluída, retrovisores e estado principal do veículo.

Nenhum VIN, token, credencial, localização ou identificador da conta é incluído nesse contrato.
